sidebar_label: HStack

A view that arranges its subviews in a **horizontal line**.

``` ts
    HStack({ alignment: cTop, spacing: 10 })(
        ForEach([1, 2, 3, 4, 5])(item =>
            Text('item' + item)
        )
    )
```
