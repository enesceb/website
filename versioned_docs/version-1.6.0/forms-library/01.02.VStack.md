sidebar_label: VStack
---

A view that arranges its subviews in a **vertical line**.

``` ts
    VStack({ alignment: cTop, spacing: 10 })(
        ForEach([1, 2, 3, 4, 5])(item =>
            Text('item' + item)
        )
    )
```
